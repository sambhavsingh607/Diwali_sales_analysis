# 🪔 Diwali Sales Analysis

## 📌 Project Overview
This project focuses on analyzing Diwali sales data using Python to understand customer purchasing behavior during the festive season. The analysis helps identify key customer segments, high-performing states, popular product categories, and spending patterns.

The project uses data cleaning, data preprocessing, exploratory data analysis (EDA), and visualization techniques to derive meaningful business insights.

---

# 📊 Objectives
- Clean and preprocess raw sales data
- Analyze customer demographics
- Identify top-performing states and occupations
- Understand spending behavior based on gender and age
- Discover most sold product categories
- Generate business insights for better marketing strategies

---

# 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

# 📂 Dataset Information
The dataset contains Diwali sales records including:
- Gender
- Age Group

- Occupation

- Orders
- Amount Spent
- Marital Status

# ⚙️ Project Workflow

## 1️⃣ Data Collection
- Imported Diwali Sales dataset using Pandas.

## 2️⃣ Data Cleaning
Performed several preprocessing steps:

- Handled missing values
- Converted data types
- Removed blank spaces

## 3️⃣ Exploratory Data Analysis (EDA)
Analyzed:

## 4️⃣ Data Visualization
Used Matplotlib and Seaborn for:
- Count plots

---

# 📈 Key Insights
- Female customers purchase more products compared to male customers.
- Customers aged between 26–35 years are the most active buyers.
- Most orders come from Uttar Pradesh, Maharashtra, and Karnataka.
- Married customers have higher purchasing power.
- Customers working in IT, Healthcare, and Aviation sectors spend more.
- Food, Clothing, and Electronics are the most popular product categories.

---

# 📌 Conclusion
The analysis shows that married women aged 26–35 years from Uttar Pradesh, Maharashtra, and Karnataka working in IT, Healthcare, and Aviation sectors are more likely to purchase products during Diwali, especially from Food, Clothing, and Electronics categories.

These insights can help businesses improve targeted marketing strategies and customer engagement.
